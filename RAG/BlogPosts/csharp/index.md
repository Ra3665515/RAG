# What is C?

C# (سي شارب) هي لغة برمجة حديثة من مايكروسوفت تم إطلاقها عام 2000.

# مميزات C#

- لغة موجهة للكائنات (Object-Oriented)
- آمنة من حيث النوع (Type-safe)
- سهلة التعلم للمبتدئين
- تدعم البرمجة غير المتزامنة (Async/Await)
- تستخدم في تطوير تطبيقات Windows وألعاب Unity
- لها مكتبة ضخمة (.NET Framework)
- تدعم LINQ للاستعلام عن البيانات
- قوية في إدارة الذاكرة تلقائياً (Garbage Collection)

C# تختلف تماماً عن لغة C القديمة، فهي أكثر حداثة وأماناً.
